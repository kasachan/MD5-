■はじめに

　このたびはダウンロードして頂きありがとうございます。
　MD5チェックツールは手軽に MD5 の計算ができるアプリケーションです。

　MD5はハッシュと言って、一方向関数によって割り出された
　ファイル固有の番号となり同じハッシュのファイルは作るのが難しいことから、
　ファイルが壊れていないか確認するのに使われています。


■インストール方法

　セットアップ版ご利用の方は
　インストーラーに従ってインストールしてください。

　ポータブル版ご利用の方はインストールの必要ありません。
　任意のフォルダーに展開してご利用ください。※

※USBフラッシュメモリーやネットワークドライブにコピーして利用することができます。


■アンインストール方法

　セットアップ版ご利用の方は
　スタートメニューもしくはコントロールパネルからアンインストールしてください。

　ポータブル版ご利用の方は
　展開先のフォルダーごと削除してください。

　レジストリ等は利用していませんが、
　Windows のプリフェッチ機能に設定が残される可能性があります。


■使い方

　セットアップ版ご利用の方は
　デスクトップのショートカット、もしくは送るメニューにあります
　「MD5チェックツール」から実行してください。

　ポータブル版ご利用の方は
　展開先のフォルダーから「MD5チェックツール.exe」を実行します。

　※Windows 98 対応版は「MD5チェックツール (9x).exe」からになります。


■ファイルの開き方

　右下の「開く(O)」ボタンから開くことができます。

　クリックしますと開く画面が出てきますから
　MD5 の計算したいファイルを開きます。※

　※複数のファイル開く際は Ctrlキー押した状態でクリックします。


　ファイルを開きますと MD5 の計算が始まります。

　計算後は大半のファイルであればすぐに計算が終わりますが
　大きなファイル (動画や ISOファイルなど) の計算は多少時間掛かることがあります。
　計算中に表示される残り時間参考にお待ちください。


■MD5の中断しかた

　大きなファイルの計算中に中断することができます。
　Windows Update やウイルスチェック、スワップ等が起きて、
　PC が重たくなった場合にお使いください。

　→一時停止　　後で計算の再開ができます。
　→キャンセル　他のファイルを計算したり MD5チェックツールを終了するときに使います。


■MD5のコピーしかた

　「コピー(C)」ボタンからクリップボードにコピーできます。


■MD5の比較しかた

　「ここに MD5 を入力します。」のエディットボックスに入力します。
　正確に 32文字入力すると MD5 の比較が始まります。

□クリップボードの貼り付け
　入力が大変な方はクリップボードから貼り付けることができます。
　「貼り付け(V)」ボタン押すと隣のエディットボックスに入力されます。

□MD5の比較結果
　MD5 が一致するとステータスアイコンが「チェックマーク」に変わります。
　このように表示されたらファイルは正常です。ご安心ください。

　反対に MD5 が一致していないとステータスアイコンが「×マーク」に変わります。
　この場合はファイルが壊れているか、MD5 を打ち間違えている可能性があります。※

　※Webサイトからコピー&ペーストした場合でも、
　　MD5 が 1文字でも欠けてしまうとダメですからよく確認しましょう。


■MD5ファイルの保存しかた

　右下の「保存(S)」ボタンから保存ができます。

　クリックしますと名前を付けて保存画面が出てきますから
　名前を入力して保存してください。

　このファイルは通常のテキストファイルとなっており、
　(ファイルリストの)ファイルごとに MD5 の結果が保存されます。


　また保存する際は名前を付けて保存画面から、
　文字コードの指定ができますからうまくご活用ください。 ※

　※現在のバージョンは次の3つが指定できます。
　・Shift_JIS (初期値)
　・Unicode
　・UTF-8 ※

　※ファイルの先頭に BOM はつけない仕様となりますから、
　　BOM に依存しているアプリやツールでご利用になる際はご注意ください。

　□バイトオーダーマーク (BOM)
　　https://ja.wikipedia.org/wiki/%E3%83%90%E3%82%A4%E3%83%88%E3%82%AA%E3%83%BC%E3%83%80%E3%83%BC%E3%83%9E%E3%83%BC%E3%82%AF


■オプション画面

□オプション画面の表示しかた

　右下の「オプション」ボタンから表示できます。
　その他にも右クリックのメニューから表示することができます。


□オプションの項目について

　→小文字で MD5 表示する。

　　このオプションにチェック入れると MD5 の計算結果が小文字で表示されます。
　　MD5チェックツールは MD5の比較で大文字と小文字の区別はしていませんから、
　　好みによりチェック入れてください。


　→非アクティブ時は半透明ウインドウにする。

　　このオプションは、他のウインドウを操作していて、
　　MD5チェックツールのウインドウが非アクティブになるとウインドウが半透明になります。
　　ただし、ファイルリストやバージョン情報などのダイアログが表示されてると、
　　半透明になりませんからご注意ください。


　→MD5ファイルは通常のファイルとして開く。

　　通常は、MD5ファイル開く際に、MD5ファイルからファイルが展開されるのですが、
　　このオプションはファイルの展開が無効にできます。
　　意味のないオプションに思いますが（＾＾；
　　MD5ファイル自身の MD5 が気になる方はお使いください。


　→起動時にファイルリストを表示する。

　　このオプションにチェック入れると次回の起動時から
　　ファイルリストが表示されます。


　→ファイルキャッシュは無効にして計算する。

　　通常はファイルが読み書きされると Windows により、
　　ファイルの内容がメモリにキャッシュされるようになっています。
　　このオプションは上記のキャッシュは無視して、ディスクから直接読み取れますから、
　　ファイルが正常に保存されているか確認することができます。
　　(外付けHDDのバックアップ後に推奨)
　　その他にもディスク本来の速度で読み書きできますからベンチマーク等速度の測定にも使えます。


　→複数のファイル計算時に描画処理を減らす。(規定でONにしています)

　　通常は複数のファイル計算時に次のファイルへ切り替わると
　　メインウインドウ下にあるファイルの情報が更新されるようになっています。
　　しかし、小さいファイルがたくさんフォルダー等では、
　　1秒に数十、または百回近くの書き換えとなり描画処理もかなりのものになるかと思います。
　　このオプションは、その描画処理が軽減できるようになり、
　　先ほどのファイル切り替わり後ではなく 500ミリ秒 ずつファイルの情報が更新されるようになります。

　　ファイルのサイズにもよりますが上記の設定で 1.2〜3倍弱 ※ 高速化します。
　　特に描画がボトルネックになりやすい、小さいファイルや仮想マシン、古いPCに効果があります。


　※小さいファイルが多い LibreOffice 4.1 より測定しました。
　　→6,383ファイル、358MB (375,544,542 バイト)
　　　いずれもオプションからディスクキャッシュ無効です。

　　・HDD:　131% 高速化 (29秒 中 22秒 に短縮)
　　　→WD Blue WD10JPVX　1TB より測定

　　・SSD:　220% 高速化 (11秒 中  5秒 に短縮)
　　　→Intel SSD 330 Series　60GB より測定

　　・RAM:　250% 高速化 ( 5秒 中  2秒 に短縮)
　　　→Buffalo RAMDISK より測定


　　測定環境について
　　・OS : Windows 8 Pro (64ビット)
　　・CPU: Intel Core i5 3550 (3.3GHz, 4コア)
　　・RAM: DDR3-1333 8GB (4GB×2)


　→フォルダー展開時は隠しファイルも追加する。

　　このオプションにチェックを入れると、
　　フォルダー内にある隠しファイルやシステムファイルが追加できるようになります。


　→MD5ファイルとして開くファイルの設定

　　このオプションは MD5ファイルとして開くファイルの設定ができます。
　　ファイルを追加するにはエディットボックスの最後に区切り文字の「;」を追加して、
　　続けて MD5ファイルとして開くファイル名を入力します。


■ファイルリストの使い方

□ファイルリストの表示しかた

　右クリックのメニューから「ファイルリストの表示」、
　もしくはオプションの設定から起動時に表示させることができます。
　上記の他にも複数のファイルを開いた時や
　MD5ファイルを計算してるときに表示するようになっています。


□MD5計算する

　ファイルリストから計算するには右上の計算開始をクリックします。
　計算後はメイン画面と同じ動作になります。

　計算開始ボタン　…計算中は「一時停止」/「再開」
　クリアボタン　　…計算中は「キャンセル」


□ファイルを追加する

　ファイルリストにファイルを追加します。
　ファイルの追加後も続けてファイルの追加ができますから
　複数のファイルで計算したい方はうまく活用してください。

→ファイルの追加しかた

　右のオプションにある「ファイルから開く」にチェック入れて、
　開くボタンを押しますと開く画面が出てきますから、
　ファイルリストに追加したいファイルを選んでください。

　※複数のファイル開くには Ctrlキー押した状態でクリックします。


□フォルダーから追加する

　ファイルリストにフォルダー内のファイルをまとめて追加できます。
　上記のように追加するのが大変な方はこちらのオプションがお勧めです。

→フォルダーからの追加しかた

　右のオプションにある「フォルダーから追加」にチェック入れて
　開くボタンを押しますとフォルダーの選択画面が出てきます。
　ファイルリストに追加したいフォルダーを選んで下にある OKボタン押してください。

　フォルダー開く際にサブフォルダー追加の確認メッセージが出ますから
　必要に応じて使い分けてください。

　※フォルダー内のファイルによっては表示に時間が掛ることがあります。
　　現在のバージョンはフォルダーの追加中にキャンセルできませんから、
　　ファイルが大量にある場合はご注意ください。


□MD5ファイルから開く

　ファイルリストに MD5ファイルに書かれてるファイルがすべて追加できます。
　対応してるファイルは当アプリケーションで作成した MD5ファイルや
　旧バージョンで作成した MD5ファイル、または、Linux の md5sum コマンドで作成したファイル等です。

→MD5ファイルから追加のしかた

　はじめに右にあるオプションから「MD5 ファイルから開く」にチェック入れます。
　続けて、開くボタン押しますと開く画面が出てきますので、
　ファイルリストに追加したい MD5ファイルを選んでください。

　この方法により追加されたファイルは MD5の計算時に MD5 の比較が同時に行われます。
　比較の結果は以下のように表示されます。

　一致　　…[チェックマーク]のアイコンで、青色の文字で表示されます。
　不一致　…[×マーク]のアイコンで、赤色の文字の文字で表示されます。


□ファイルリストから削除する

　ファイルリストから1つのファイルをリストから削除できます。
　間違えて違うファイルを開いてしまった場合にお使いください。

→ファイルリストから削除のしかた

　はじめにファイルリストから削除したいファイルを右クリックします。
　右クリックするとメニューが出てきますから
　メニューの下の方にある「リストから削除」をクリックして完了です。


□ファイルリストから保存する

　右にある保存ボタンからMD5ファイルに保存ができます。
　このファイルは通常のテキストファイルとなっており、
　(ファイルリストの)ファイルごとに MD5 の結果が保存されます。


□ファイルリストをクリアする

　右上のクリアボタン押してすべてのファイルが削除できます。
　ファイルリストをクリアする際は確認のメッセージは出ませんから
　間違えてクリアしないようにご注意ください。

■既知の不具合

□ファイルの名前に Unicode の特殊文字が含まれているとMD5ファイルが正常に保存できません。

　この不具合はMD5ファイルに Unicode が使われていないことにより起こります。
　現在は互換性の高い Shift_JIS の文字コードが使われていますが、
　Shift_JIS には存在しない Unicode の特殊文字がありますから、
　その文字がファイル名に含まれてしまうと正常に保存ができなくなります。


　→解決方法
　・Unicode の特殊文字は利用しないようにファイルの名前変更します。
　・Shift_JIS 以外の文字コードで保存します。※

　※名前を付けて保存画面の下にあります「文字コードの設定」から設定ができます。


□Windows 8　64bit版 の regedit.exe は正常に計算できません。

　おそらく Windows の不具合かと思いますが、
　ファイルの読み込みが 155KB 中 130KB の場所でストップしてしまいます。
　現状は解決方法がありません ※ から、計算はキャンセルするようにお願いします。

※他のフォルダーへコピーすると正常に計算できるようですが現実的ではないと思います。（＾＾；
